### 1) 카드추가탭

- [ ] 카드번호
  - [x] xxxx-xxxx-oooo-oooo의 형식을 따른다.
  - [x] 뒤의 여덟자리는 감춤형식을 사용한다.
  - validation
    - [ ] 16자리로 이루어져야한다.
    - [x] 숫자만 입력가능하다.
- [ ] 만료일
  - [x] MM/YY 형식을 따른다.
  - [ ] validation
    - [ ] 날짜의 형식이어야한다.
    - [ ] MM은 01부터 12까지 가능하다.
- [ ] 소유자 이름
  - [ ] 선택사항이다.
  - [ ] 이름을 입력하지 않을시 Default값(name)을 준다.
  - [ ] validation
    - [x] 이름은 30자이하여한다.
- [ ] 보안 코드
  - [x] 세자리의 감춤형식을 사용한다.
  - [x] ?버튼을 누르면 도움말 모달을 띄어준다.
  - [ ] validation
    - [x] 숫자여야한다.
    - [x] 3자리여야한다.
- [ ] 카드비밀번호
  - [x] 앞의 두자릿수를 감춤형식으로 입력받는다.
  - [ ] validation
    - [x] 숫자여야한다.
    - [x] 두자리여야한다.
- [x] 모든 입력값은 카드 예시와 동기화 시킨다.
- [ ] 카드를 클릭했을시 색깔 변경이 가능하다.
- [ ] 빈 값이나 채워지지 않은 값이 있을 시 제출 불가
